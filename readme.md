# Playlist Partagé
Playliste collective comme spotify pour voter la musique qui va passer 
---
Playlist bouge en fonction des votes
## Outils
* lecteur(pygame.mixer)
* liste de musique (mp3)
* utilisateur 
* sqlite
* python 
* tableur 
(+ peut mettre un mode premium pour avoir plus de vote contre de la moulla)
